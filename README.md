# Task Manager API 🚀

## 📌 Project Overview
A FastAPI-based backend application for managing tasks with authentication.

---

## 🛠️ Tech Stack
- FastAPI
- SQLAlchemy
- MySQL
- JWT Authentication

---

## ✨ Features
- User Registration & Login
- JWT Authentication
- Task CRUD Operations
- Logging
- Background Scheduler

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
